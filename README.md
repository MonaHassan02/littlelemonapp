Little Lemon Restaurant - Table Reservation App
🍋 Project Overview
This project is a modern, responsive web application for Little Lemon, a family-owned Mediterranean restaurant. The primary feature is a multi-step functional booking system that allows users to reserve a table, choose their occasion, and receive instant confirmation.

This application was developed as part of the Meta Front-End Developer Professional Certificate Capstone.

🚀 Features
Dynamic Table Booking: Users can select dates, times, and occasions (Birthdays, Anniversaries, etc.).

Real-time Validation: Includes HTML5 and JavaScript validation to ensure user data (like Full Name and Guest count) is correct before submission.

Responsive Design: Fully optimized for Desktop, Tablet, and Mobile views.

Interactive UI: A multi-step process including a "Confirm Your Details" summary card and a success confirmation page.

Unit Tested: Core logic and UI components are verified using Jest and React Testing Library.

🛠️ Technologies Used
Frontend: React.js

Routing: React Router v6

Styling: CSS3 (Flexbox & Grid)

Testing: Jest & React Testing Library

Fonts: Google Fonts (Markazi Text & Karla)
